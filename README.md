## Installation

Install the dependencies and devDependencies.
- Open terminal
- Need to create a project on firebase and add the details of your firebase in App.js
- Install npm

```sh
#In terminal
    git clone https://github.com/anandhu720/youtube-clone.git
    cd youtube-clone
    npm start
    
```
